Crear un repositorio git llamado Book:
  git init Book

O bien, desde un directorio git2 (creado anteriormente), se crea un repositorio git desde dentro de git2:
  git init .

git add .
git commit -m "Comentario de Confirmacion"


Como cambiar el comentario de un commit:
  git commit --amend -m "Nuevo Comentario"

git status

Lista la branca:
  git branch

Crea una rama nueva:
  git branch segunda

Cambia a la Segunda branca:
  git checkout segunda
